# POWER BI/TABLEAU PROJECT (Power BI: Data Professional Survey Analysis / Tableau: Netflix Dashboard Visualization) 

## Project Overview

This repository contains two data analytics projects focused on deriving insights from real-world datasets.
‎
‎The Power BI project analyzes survey responses from 600–700 data professionals to uncover trends in salary, tools, and work-life balance.
‎The Tableau project visualizes Netflix content data, highlighting trends in genres, countries, and content distribution.
