# POWER BI/TABLEAU PROJECT (Power BI: Data Professional Survey Analysis / Tableau: Netflix Dashboard Visualization) 

## Project Overview

This repository contains two data analytics projects focused on deriving insights from real-world datasets.
‎
‎The Power BI project analyzes survey responses from 600–700 data professionals to uncover trends in salary, tools, and work-life balance.
‎The Tableau project visualizes Netflix content data, highlighting trends in genres, countries, and content distribution.

## Data Sources

‎Survey data collected via LinkedIn, X, and other platforms
‎Public Netflix dataset

## Tools

- Power BI (Power Query, Dashboarding)

- Tableau (Visualization & Mapping)

## Data Cleaning & Preparation

- Removed blank columns

- Split and formatted columns using delimiters

- Standardized data for analysis


## Data Analysis

### Power BI Dashboard

- Average Salary by Job Title

- Most Used Programming Languages

- Country-Based Salary Insights

- Work-Life Balance (Gauge Chart)

- Gender Salary Comparison
